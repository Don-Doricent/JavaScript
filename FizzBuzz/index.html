FizzBuzz Exercise

This is normally Psuedo Code.

  1. When a user inputs a number, loop from 1 to the entered number.
  2. If the current number is divisible by 3 then print "Fizz"
  3. If the current number is divisible by 5 then print "Buzz"
  4. If the current number is divisible by 3 AND 5 then print "FizzBuzz"
  5. Otherwise print the current number
